# NYC Air Quality Analysis

This project explores air quality trends across New York City using EPA data.  
The analysis includes yearly and seasonal breakdowns, city comparisons, and  
public health implications related to emissions and asthma rates.

**Files included:**
- `NYC Air Quality Analysis.docx`: Final written report and insights
- `NYC Air Quality Dataset.xlsx`: Cleaned dataset used for analysis
- `Tableau Public Viz.docx`: Visualizations and Tableau link summary
